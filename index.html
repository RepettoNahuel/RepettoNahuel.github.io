Login

Sistema web de autenticación y registro de usuarios implementado con HTML para la estructura de la página, CSS para el diseño, JavaScript para la interactividad en el cliente, y PHP para la interacción con la base de datos. La interfaz incluye campos para el nombre de usuario y la contraseña, así como botones de "Ingresar" y "Registrarse". Al hacer clic en uno de estos botones, el sistema verifica la existencia de los datos en la base de datos. Si los datos existen y se intenta iniciar sesión, se muestra un mensaje de "Ingreso Exitoso"; de lo contrario, se muestra una advertencia. Si se intenta registrar un usuario que ya existe, también aparecerá una advertencia; si no existe, se muestra un mensaje de "Registro Exitoso" y se redirige a la página de inicio de sesión. Puede probar con los usuarios “user1”, “user2” o “user3”, con la contraseña “1234”, o también puede crear uno nuevo.

Lista de Tareas

Lista de tareas interactiva utilizando HTML, CSS y JavaScript. La aplicación permite agregar tareas a través de un campo de entrada y un botón, que luego se muestran en la lista. Cada tarea cuenta con opciones para marcarla como completada o eliminarla. Este proyecto muestra que se pueden lograr aplicaciones web dinámicas e interactivas utilizando tecnologías front-end comunes. Además, para evitar que las tareas desaparezcan al refrescar la página o salir de esta, utilice conceptos básicos de PHP para conectar la lista de tareas con una base de datos que se actualiza al interactuar con esta.
