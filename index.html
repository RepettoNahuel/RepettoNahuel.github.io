Recetas

Una página web interactiva utilizando HTML, CSS y JavaScript, donde presento una colección de recetas culinarias. Hice uso del DOM para ofrecer una experiencia interactiva y atractiva para los usuarios que buscan inspiración culinaria. Cada receta se muestra como una imagen en un recuadro, que al pasar el cursor sobre una la misma, aparece el nombre de la receta. Al hacer clic, se despliega una lista de sus ingredientes. Cuando el cursor deja de estar sobre la imagen, vuelve a mostrarse solo la foto.
